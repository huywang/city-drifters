# 🏙️ 都市浮生记 - 漂泊人生模拟器

> 每一个在大城市漂泊的灵魂，都在寻找一个答案。

一款关于中国大城市漂泊者的人生模拟网页游戏。

## 🎮 游戏特色

### 丰富的出身和城市
- **6种出身背景**：985计算机毕业、文科毕业生、小镇做题家、中年失业者、大专技术人、海归留学生
- **6个大城市**：北京、上海、深圳、杭州、广州、成都（每个城市有专属事件和特色）

### 海量内容
- **170+ 生活事件**：覆盖职场、住房、社交、健康、财务、婚姻、育儿、投资、人生重大选择
- **36种人生结局**：从财务自由到过劳死，从大厂高管到返乡青年
- **50个成就**：记录你人生的每一个里程碑
- **每日挑战系统**：每天不同的目标，完成获得奖励
- **跨游戏统计**：记录你的所有游戏历程
- **社会新闻事件**：烂尾楼、P2P爆雷、教培双减、疫情封控、AI替代焦虑、考公热、电信诈骗、天价彩礼等

### 深度系统
- **投资理财系统**：股票、基金、加密货币、P2P，每一样都有风险
- **人际关系系统**：家人、朋友、恋人、同事，四维度关系追踪
- **事件链系统**：某些选择会触发后续事件
- **技能爱好**：健身、阅读、烹饪、音乐、摄影，培养你的生活方式
- **月度活动选择**：每个月选择你的重点（工作/休息/学习/社交/锻炼），掌控人生节奏
- **季节系统**：春夏秋冬影响你的心情和健康，还有季节专属事件
- **主题切换**：支持深色/浅色模式

### 2025-2026时代元素
- AI替代焦虑
- 躺平vs内卷
- 消费降级
- 远程办公
- FIRE财务自由计划
- 数字游民生活

## 🎯 玩法

1. 选择你的出身背景
2. 选择你要漂泊的城市
3. 每个月选择你的活动重点（工作/休息/学习/社交/锻炼）
4. 面对各种生活事件，做出你的选择
5. 你的选择影响六项属性：💰金钱、❤️健康、😊心情、🧠智力、👥人脉、✨魅力
6. 不同的属性组合导向不同的人生结局

## 🕹️ 操作

### 键盘快捷键
- **空格/Enter**：下个月
- **1-5**：选择月度活动
- **M**：打开/关闭菜单
- **Esc**：关闭菜单/弹窗

### 移动端
- **左滑**：下个月
- **右滑**：打开菜单

## 🛠️ 技术栈

- 纯前端实现（HTML5 + CSS3 + JavaScript）
- 无需后端，可直接部署到 GitHub Pages
- 响应式设计，支持移动端
- 本地存储（localStorage）保存游戏进度
- 3个存档槽位，支持导入/导出存档
- 无障碍支持：高对比度模式、减少动画、字体大小调整
