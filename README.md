# 🏙️ 都市浮生记 - 漂泊人生模拟器

> 每一个在大城市漂泊的灵魂，都在寻找一个答案。

一款关于中国大城市漂泊者的人生模拟网页游戏。

## 🎮 游戏特色

- **6种出身背景**：985计算机毕业、文科毕业生、小镇做题家、中年失业者、大专技术人、海归留学生
- **6个大城市**：北京、上海、深圳、杭州、广州、成都
- **49个生活事件**：覆盖职场、住房、社交、健康、财务、人生重大选择
- **14种人生结局**：从财务自由到过劳人生，每个选择都影响命运
- **网络热梗**：充满黑色幽默和时代特征的故事情节

## 🎯 玩法

1. 选择你的出身背景
2. 选择你要漂泊的城市
3. 每个月面对各种生活事件，做出你的选择
4. 你的选择影响六项属性：💰金钱、❤️健康、😊心情、🧠智力、👥人脉、✨魅力
5. 不同的属性组合导向不同的人生结局
